=============================
    TEMPERATURE CONVERTER
=============================

PROJECT TITLE:
Super Premium Temperature Converter

DESCRIPTION:
This is a modern and responsive Temperature Converter
web application developed using HTML, CSS, and JavaScript.
The application allows users to convert temperatures
between Celsius (°C), Fahrenheit (°F), and Kelvin (K)
with an attractive user interface and instant results.

-----------------------------------------
FEATURES
-----------------------------------------

✓ Convert Celsius to Fahrenheit
✓ Convert Celsius to Kelvin
✓ Convert Fahrenheit to Celsius
✓ Convert Fahrenheit to Kelvin
✓ Convert Kelvin to Celsius
✓ Convert Kelvin to Fahrenheit
✓ Input Validation
✓ Modern Glassmorphism Design
✓ Responsive Layout
✓ Animated Background Effects
✓ Interactive UI
✓ Enter Key Support

-----------------------------------------
TECHNOLOGIES USED
-----------------------------------------

1. HTML5
2. CSS3
3. JavaScript

-----------------------------------------
HOW TO RUN
-----------------------------------------

1. Download or create all project files.
2. Place index.html, style.css,
   script.js and README.txt
   in the same folder.
3. Open index.html in a web browser.

OR

1. Open project folder in VS Code.
2. Install Live Server extension.
3. Right click on index.html.
4. Click "Open with Live Server".

-----------------------------------------
HOW TO USE
-----------------------------------------

1. Enter temperature value.
2. Select the source unit (From).
3. Select the target unit (To).
4. Click Convert Temperature.
5. View the converted result instantly.
